# programacion_de_videojuegos
<p>
<strong>Platform Peril</strong> es un minijuego de plataformas donde la precisión y los reflejos son la clave. 
El jugador debe avanzar por un recorrido lleno de plataformas móviles y obstáculos, manteniendo el equilibrio y evitando caídas. 
Cada salto y cada movimiento cuentan: un error puede significar empezar de nuevo.  

Este proyecto busca recrear la esencia de los clásicos minijuegos de <em>Mario Party</em>, ofreciendo una experiencia rápida, divertida y desafiante. 
Ideal para partidas cortas, pero con la tensión suficiente para mantenerte al borde del asiento.
</p>
